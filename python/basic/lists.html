Python Lists

In this article you will learn what's a List

What's a List?

A List can be taken as a box in which you have a thing for example a photo of a house
or an old item or anthing

How to create a List

A group of elements is called a list in python. Python lists are very similar to an array but the main difference is, an array can store only one type of element whereas, a list can store different types of elements. List are mutable (changeable).
Lists are dynamics which means size is not fixed. Lists are represented using a square bracket[ ], separated by commas. We initialise a list with square brackets, and create it the same way we create variable. Give the list a name, followed by an equal sign and square brackets. Add elements in the list separated by commmas separated by commas

# This is a List called "animalss" containing a list of animalss.
animals #gives ["moose", "dog", "cat", "turtle"]

Here we have declared a List called animalss
We can add another animals to the list by using the append() function. Suppose we want to add a donkey to the end of list. For that, we would simply write:

animals.append("donkey") animals #gives ["moose", "dog", "cat", "turtle", "donkey"]

animals now contains the donkey!

To remove items off of a list, use the remove() function

animalss.remove("cat") animalss = ["moose", "dog", "turtle", "donkey"]

animals now doesn't contain the cat!

How to access items in a List

We can access items in a list using their index. Python lists indexes start at 0, so the first element of the list.
will be located at position 0. To access an element of the list via index, write the name of the list followed by square brackets, and in these brackets write the index position. In our case, with the animals list, to access the elements by index we would write:

animals ["moose", "dog", "cat", "turtle"]
animals[0] #gives "moose"
animals[1] #gives "dog"
animals[2] #gives "cat"
animals[3] #gives turtle"

Yoo can access the elements of the list by using negative numbers. So animas[-1] would give you the last element of the list, animalss[-2] the next to last element and so on:

animals ["moose", "dog", "cat", "turtle"]
animals[-4] #gives "moose"
animals[-3] #gives "dog"
animals[-2] #gives "cat"
animals[-1] #gives turtle"

How to replace items in a List

Finally, you can use indexes to replace items in a list. Suppose I wanted to replace "dog" with parrot, in my animals list. Dog is the 2nd element of the list, so is at index animalss[1]. (Rememember that indexes start at 0). To replace "dog" with parrot I would simplty write:

animals[1] = "parrot

And printing out the list would yield:

print(animals) #yields ["moose", "parrot", "cat", "turtle"]

so we have successfully replaced the dog with the parrot in our list!.

case-sensitive

Remember that Python is case sensitive so Apple is NOT the same as apple.

How to name a List

do and don't to name a List in python are the same as the dos and donts to name variables in python

  • List name must not start with a number(1name, 23name)
  • List name must start with latters or underscore(name, _name)
  • List name can contain number but after a latter(name1, n1ame)
  • List name must not have space in between(my age)

multi-word naming

To make it clear to read a List and other variables name, it is best that programmers follow the PEP 8 naming convention.
This convention specifies the best naming practices programmers must follow when writing python code.
